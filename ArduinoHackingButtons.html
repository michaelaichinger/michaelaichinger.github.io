3.6 Hacking Buttons

Bei diesem Projekt wurde eine Stirnlampe auseinandergebaut und mit Arduino verkabelt.
Anschließend wurde ein kurzes Programm geschrieben, damit das Licht in einen bestimmten Rhythmus blinkt.

const int optoPin = 2;

void setup() {
    pinMode (optoPin, OUTPUT);

}

void loop() {
    digitalWrite (optoPin, LOW);
    delay (100);
    digitalWrite (optoPin, HIGH);
    delay (200);
    digitalWrite (optoPin, LOW);
    delay (300);
    digitalWrite (optoPin, HIGH);
    delay (400);
    
}
